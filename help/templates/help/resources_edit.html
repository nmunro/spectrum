Help: Edit Resource

Editing a resource is a similar process to adding a resources, from the list of resources, select the resource you wish to edit, and click the "edit" button.

You will be presented with the same form, like with adding a resource, cancelling the form will return the user to the list of resources, saving the form will update its information.
