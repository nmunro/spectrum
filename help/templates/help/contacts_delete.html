Help: Delete Contact

Deleting a contact is a two step process, from the list of contacts, select the contact you wish to delete and click the delete button.

The user will be asked to confirm the deletion of a contact.

The user may cancel this to prevent the deletion. Confirming the delete will return the user to the list of contacts.
