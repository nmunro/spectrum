Help: Add Location

Adding a location to an organisation requires at least three required field(s) and two optional field(s). As seen in the screenshot below. Fields indicated with an asterisk (*) are required and must be provided.

Any given user may be responsible for managing multiple organisations and as such when creating a location they must select the organisation the location will belong to. If you only manage one organisation you will only see one in the drop down.

To create a location one must select the "Create New" button from the previous page. If there are any errors in the form, these will be displayed with a warning. To cancel creating a location click the red cancel button, this will return the user to the list of locations.

Saving the form will likewise return the user to the list of locations.
