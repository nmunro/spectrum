Help: Resources

Resources are a type of object that an organisation may make available. This can be a document, object, etc, because a resource is a rather vague thing to define, it is left up to the organisation admin to accurately describe the nature of the resource.

From the interface below, you will be able to view, add, edit, and delete resources from an organisation.
