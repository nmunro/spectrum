Help: Delete Schedule

Deleting a schedule is a two step process, from the list of schedules, select the schedule you wish to delete and click the delete button.

The user will be asked to confirm the deletion of a schedule.

The user may cancel this to prevent the deletion. Confirming the delete will return the user to the list of schedules.
