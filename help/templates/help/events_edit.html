Help: Edit Event

Editing an event is a similar process to adding an event, from the list of events, select the event you wish to edit, and click the "edit" button.

An event requires both a contact and a location which must be created prior to creating an event.

You will be presented with the same form, like with adding an event, cancelling the form will return the user to the list of events, saving the form will update its information.
