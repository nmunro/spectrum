Help: Add Schedule

Introduction

The Schedule Wizard allows you to create recurring events with various configurations. Below are the steps for each scheduling option: Yearly, Monthly, Weekly, and Daily. Each option comes with different permutations and end conditions to meet your scheduling needs.

Yearly Schedule

To create a yearly schedule:

  1. Set the frequency to Yearly.
  2. Select the type of yearly recurrence:
    • By Date in Month: Choose the specific month and day.
    • By Day in Month: Choose the specific occurrence of a weekday (e.g., 3rd Friday) in a particular month.
  3. Set the end condition:
    • Never Ends: The event will continue indefinitely.
    • Ends After: Specify the number of occurrences after which the event ends.
    • Ends on Date: Select a specific end date for the schedule.
  4. Click Save to create the schedule.

Monthly Schedule

To create a monthly schedule:

  1. Set the frequency to Monthly.
  2. Select the type of monthly recurrence:
    • By Date in Month: Choose a specific day of the month.
    • By Day in Month: Choose the nth occurrence of a weekday within the month (e.g., 2nd Tuesday).
  3. Set the interval (e.g., every 1 month, every 2 months).
  4. Set the end condition:
    • Never Ends: The event will continue indefinitely.
    • Ends After: Specify the number of occurrences after which the event ends.
    • Ends on Date: Select a specific end date for the schedule.
  5. Click Save to create the schedule.

Weekly Schedule

To create a weekly schedule:

  1. Set the frequency to Weekly.
  2. Select the interval (e.g., every 1 week, every 2 weeks).
  3. Select the days of the week when the event should occur (e.g., Monday, Wednesday, Friday).
  4. Set the end condition:
    • Never Ends: The event will continue indefinitely.
    • Ends After: Specify the number of occurrences after which the event ends.
    • Ends on Date: Select a specific end date for the schedule.
  5. Click Save to create the schedule.

Daily Schedule

To create a daily schedule:

  1. Set the frequency to Daily.
  2. Select the interval (e.g., every day, every 2 days).
  3. Set the end condition:
    • Never Ends: The event will continue indefinitely.
    • Ends After: Specify the number of occurrences after which the event ends.
    • Ends on Date: Select a specific end date for the schedule.
  4. Click Save to create the schedule.

End Conditions

Regardless of the frequency you choose, setting the end condition is crucial to ensure the schedule meets your requirements. You can choose from:

  • Never Ends: Ideal for ongoing events without a predetermined conclusion.
  • Ends After: Useful when you know the number of occurrences required (e.g., a 10-week course).
  • Ends on Date: Best for events with a definite end date (e.g., a project deadline).

Common Issues and Considerations

When configuring schedules, be mindful of the following potential issues:

  • Date Skipping: Schedules set for a specific day, like the 31st of each month, will not be generated in months that lack that day (e.g., February, April, June, September, November). For continuous monthly schedules, consider choosing the "Last Day" of each month to avoid skipping months.
  • Irregular Weekday Occurrences: When scheduling events like the "3rd Friday of each month," the schedule might skip months with fewer occurrences of that weekday. For example, in a 4-week month, this might not apply.
  • Time Zone Sensitivity: Recurrence times may vary based on the time zone of participants. Ensure that all users are aware of the time zone in which the schedule was created.
