Help: Add Resource

Adding a resource to an organisation requires at least three required field(s) and one optional field(s). As seen in the screenshot below. Fields indicated with an asterisk (*) are required and must be provided.

Any given user may be responsible for managing multiple organisations and as such when creating a contact they must select the organisation the contact will belong to. If you only manage one organisation you will only see one in the drop down.

To create a resource one must select the "Create New" button from the previous page. If there are any errors in the form, these will be displayed with a warning. To cancel creating a resource click the red cancel button, this will return the user to the list of resources.

It is important to note that a user may use the name, description, or any of the listed tags to search across, this allows for tags being used to optimise search queries, while tags are optional, it is highly recommended to add as many tags as possible to assist in discovery.
