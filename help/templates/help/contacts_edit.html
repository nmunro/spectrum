Help: Edit Contact

Editing a contact is a similar process to adding a contact, from the list of contacts, select the contact you wish to edit, and click the "edit" button.

You will be presented with the same form, like with adding a contact, cancelling the form will return the user to the list of contacts, saving the form will update its information.
